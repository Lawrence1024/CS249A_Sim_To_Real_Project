# Webots Robotics Examples

This folder contains example Scenic scenarios for robotics using the new robotics domain
with Webots simulator integration.

## Examples

- **simple_robot.scenic**: Basic robot movement and control
- **line_following.scenic**: Robot following a line using IR sensors
- **obstacle_avoidance.scenic**: Robot avoiding obstacles using ultrasonic sensors
- **maze_navigation.scenic**: Robot navigating through a maze

## Running the Examples

1. **Install Webots** and ensure Scenic is installed in Webots' Python environment
2. **Open Webots** and load the appropriate `.wbt` world file
3. **Start the simulation** - Scenic will automatically run and control the robot

## World Files

Each example has a corresponding `.wbt` world file in the `webots_data/worlds/` directory
that defines the robot, sensors, and environment for the scenario.
